# <めんそーれ>

## サイト概要
沖縄の観光グルメのおすすめ紹介、共有サイト

### サイトテーマ
地元民と観光客の中間の立ち位置から見た沖縄案内

### テーマを選んだ理由
沖縄に仕事で2年住んでいて住んだからこそ地元の人では気がつきにくいおすすめの紹介共有サイト

### ターゲットユーザ
SNSに馴染みがある10代後半〜40代

### 主な利用シーン
沖縄旅行に行こうと思った際の行き先選び、時間が余った際寄り道探し
